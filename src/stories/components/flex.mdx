# Flex

Use the `Flex` component to layout the children using [flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/).

A simple example

<Preview>
  <p>Defaults to row</p>
  <Story name="row">
    <Flex>
      <Box width={1 / 3} p={3} bg="success.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bg="warning.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bg="error.main">
        Box
      </Box>
    </Flex>
  </Story>
  <p>Set to column</p>
  <Story name="column">
    <Flex flexDirection={'column'}>
      <Box width={1 / 3} p={3} bg="success.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bg="warning.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bg="error.main">
        Box
      </Box>
    </Flex>
  </Story>
</Preview>
