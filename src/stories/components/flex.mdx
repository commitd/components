# Flex

Use the `Flex` component to layout the children using [flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/).

A simple example

<Preview>
  <p>Defaults to row</p>
  <Flex>
    <Box width={1 / 3} p={3} bg="success">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="warning">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="error">
      Box
    </Box>
  </Flex>
  <p>Set to column</p>
  <Flex flexDirection={"column"}>
    <Box width={1 / 3} p={3} bg="success">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="warning">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="error">
      Box
    </Box>
  </Flex>
</Preview>
