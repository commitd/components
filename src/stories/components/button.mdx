# Button

Buttons make common actions immediately visible and easy to perform with one click or tap. They can be used for any type of action, including navigation.

<Preview>
  <Story name="simple">
    <Button margin={2}>Hello</Button>
  </Story>
</Preview>

# Contained

This is the default style for the button, and should be used for the primary action.

<Preview>
  <Story name="default">
    <Button>Default</Button>
  </Story>
  <Story name="primary">
    <Button color="primary">Primary</Button>
  </Story>
  <Story name="secondary">

  </Story>
</Preview>

# variants

Some button variants are available and should be used for secondary buttons such as "cancel".

## Outlined

Use outlined for lower emphisis actions.

<Preview>
  <Story name="outlined default">
    <Button variant="outlined">Default</Button>
  </Story>
  <Story name="outlined primary">
    <Button variant="outlined" color="primary">
      Primary
    </Button>
  </Story>
  <Story name="outlined secondary">
    <Button variant="outlined" color="secondary">
      Secondary
    </Button>
  </Story>
</Preview>
