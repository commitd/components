# Flex

Use the `Flex` component to layout the children using [flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/).

A simple example

<Playground>
  <p>Defaults to row</p>
  <Flex>
    <Box width={1 / 3} p={3} bg="blue">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="green">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="red">
      Box
    </Box>
  </Flex>
  <p>Set to column</p>
  <Flex flexDirection={"column"}>
    <Box width={1 / 3} p={3} bg="blue">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="green">
      Box
    </Box>
    <Box width={1 / 3} p={3} bg="red">
      Box
    </Box>
  </Flex>
</Playground>
