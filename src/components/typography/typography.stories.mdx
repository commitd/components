# Headings

Use `Heading` for rendering heading html element e.g. `h1, h2, ...`. Headings should not be used as a substitute for font size changes,
they should be used in line with [best practice](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Heading_Elements). They have, by default,
responsive size settings, however the sizes can be set independently using the `fontSize` prop and selecting a responsive list of sizes from theme's
fontSizes array, `fontSize={[3,4,5]}`.

# Display

Traditional heading elements are designed to work best in the main page content.
When you need a heading to stand out, consider using a display heading: a larger, slightly more opinionated heading style.

# Text

Use `Text` for all normal text rendering, use Headings for `h1, etc`.
Text is a low-level component for setting font-size, typographic styles, margin, and color.
By default Text is a `div`, the alternative classes `Span` and `Paragraph` can be used for `p` and `span` elements.
