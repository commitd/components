# Utilities

We collect some utilities that may be useful.

## Use width hook

A hook to get the current Breakpoint range we are in using the text code.

```
value         |0px     600px    960px    1280px   1920px
key           |xs      sm       md       lg       xl
screen width  |--------|--------|--------|--------|-------->
range         |   xs   |   sm   |   md   |   lg   |   xl
```

<Canvas>
  <Story name="use Width Hook">
    {React.createElement(() => {
      const width = useWidth()
      return (
        <Box bgcolor="background.paper">
          <Typogaphy bgcolor="background.paper">
            Current breakpoint is {width}
          </Typogaphy>
        </Box>
      )
    })}
  </Story>
</Canvas>

## Copy to clipboard

A utility function to support copy to clipboard functionality, as used in [colors](/components/?path=/docs/design-system-colour--page) and [icons](/components/?path=/docs/components-icons--default-story).

<Canvas>
  <Story name="copy">
    <Button
      color="primary"
      onClick={() => {
        copyToClipboard('Copied')
      }}
    >
      Copy
    </Button>
  </Story>
</Canvas>

This should work in a dialog

<Canvas>
  <Story name="default">
    {() => {
      const [open, setOpen] = React.useState(false)
      function handleClickOpen() {
        setOpen(true)
      }
      function handleClose() {
        setOpen(false)
      }
      return (
        <div>
          <Button variant="outlined" color="primary" onClick={handleClickOpen}>
            Open dialog
          </Button>
          <Dialog open={open} onClose={handleClose}>
            <DialogTitle id="alert-dialog-title">
              {'Copy in dialog?'}
            </DialogTitle>
            <DialogContent>
              Copied in dialog{' '}
              <Button
                color="primary"
                onClick={() => {
                  copyToClipboard('Copied in dialog')
                }}
              >
                Copy
              </Button>
            </DialogContent>
            <DialogActions>
              <Button onClick={handleClose} color="primary" autoFocus>
                Close
              </Button>
            </DialogActions>
          </Dialog>
        </div>
      )
    }}
  </Story>
</Canvas>
