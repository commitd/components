# Loader

<Canvas>
  <Story name="default">
    <Loader color="secondary.dark" />
  </Story>
</Canvas>

The loading prop can be used to stop the animation and be used as a static loading indicator

<Canvas>
  <Story name="loading">
    {() => {
      const [loading, setLoading] = React.useState(false)
      return (
        <Column>
          <Loader loading={loading} />
          <Button
            variant="outlined"
            color="primary"
            onClick={() => setLoading(!loading)}
          >
            Toggle
          </Button>
        </Column>
      )
    }}
  </Story>
</Canvas>

The size can be controlled with the `size` prop.

<Canvas>
  <Story name="small">
    <Loader color="secondary.dark" size={100} />
  </Story>
</Canvas>

Omit the `color` to get the default greyscale version.

<Canvas>
  <Story name="grey">
    <Loader size={200} />
  </Story>
</Canvas>
