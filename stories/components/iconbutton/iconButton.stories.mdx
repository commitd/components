# IconButton

Extends [Material UI IconButton](https://material-ui.com/api/icon-button/)

Icon buttons can be used for common and obvious actions, where the semantics can not be misunderstood.
Or in compact forms of drawers or toolbars.

Icons are also appropriate for toggle buttons that allow a single choice to be selected or deselected,
such as adding or removing a star to an item.
