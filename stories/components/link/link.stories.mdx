# Link

Extends [Material UI Link](https://material-ui.com/components/links/)

A Link is an anchor element with extra styleing options.

<Preview>
  <Story name="default">
    <Typography>

      <Link href="javascript:;" color="inherit">
        color="inherit"
      </Link>{' '}
      <Link href="javascript:;">
        <Typography variant="body2">variant="body2"</Typography>
      </Link>{' '}
      <Link href="http://committed.io" target="_blank">
        External
      </Link>
    </Typography>
  </Story>
</Preview>

## Security

When you use `target="_blank"` with Links, we set `rel="noopener"` when linking to third party content
this prevents the new page from being able to access the `window.opener` property and ensures it runs in a separate process.
Without this, the target page can potentially redirect your page to a malicious URL. This can be overridded
(say for `noreferrer` which additionally removes the the referrer header) by supplying a `rel` prop.
