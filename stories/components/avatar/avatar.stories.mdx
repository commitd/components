# Avatar

Extends [Material UI Avatar](https://material-ui.com/components/avatars/).

The Avatar should be used for profile images. If an image is not available intials can be used.

<Canvas>
  <Story name="default">
    <Flex>
      <Avatar m={1} alt="John Smith" src="https://i.pravatar.cc/40" />
      <Avatar
        m={1}
        alt="John Smith"
        src="https://i.pravatar.cc/60"
        size="60px"
      />
    </Flex>
  </Story>
</Canvas>

## Letter based

To just use initials:

<Canvas>
  <Story name="letter based">
    <Flex>
      <Avatar m={1} bgcolor="primary.main">
        H
      </Avatar>
      <Avatar m={1} bgcolor="success.dark">
        N
      </Avatar>
      <Avatar m={1} bgcolor="warning.main">
        OP
      </Avatar>
      <Avatar m={1} bgcolor="red">
        H
      </Avatar>
      <Avatar m={1} bgcolor="#AB34FA">
        PP
      </Avatar>
    </Flex>
  </Story>
</Canvas>

## Icons

In addition, icons are supported and can be used for intro banners.

  <Story name="icons">

      </Avatar>

      </Avatar>
    </Flex>
  </Story>
</Canvas>
