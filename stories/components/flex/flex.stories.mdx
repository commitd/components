# Flex

Extends [Material UI Box](https://material-ui.com/components/box/) setting the display to `flex`.

Use the `Flex` component to layout the children using [flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/).

A simple example

<Canvas>
  <p>Defaults to row</p>
  <Story name="default">
    <Flex>
      <Box width={1 / 3} p={3} bgcolor="success.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bgcolor="warning.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bgcolor="error.main">
        Box
      </Box>
    </Flex>
  </Story>
  <p>Set to column</p>
  <Story name="column">
    <Flex flexDirection={'column'}>
      <Box width={1 / 3} p={3} bgcolor="success.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bgcolor="warning.main">
        Box
      </Box>
      <Box width={1 / 3} p={3} bgcolor="error.main">
        Box
      </Box>
    </Flex>
  </Story>
</Canvas>

## Row and Column

For convenience we also export `Row` and `Column` components as aliases for `flexDirection="row"` and `flexDirection="column"`.

<Canvas>
