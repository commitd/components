# Icon

We recommend using the Sharp Material-UI icons directly from `@material-ui/icons` and setting up a project icons file to allow easy changing of all semantically equivalent icons.
This will help to improve maintainability and reduce bundle size through tree shaking.

Click to copy the name. Then use `import {Name} from '@material-ui/icons'`
